# Brain — оперативная память проекта для людей и агента

Короткие, проверяемые документы: **как устроено**, **как писать код**, **что нельзя ломать**. Детальные ТЗ и длинные планы по-прежнему в [`docs/`](../docs/).

| Папка | Назначение |
|--------|------------|
| [`core/`](./core/) | Стек, границы системы, доменные сущности, ограничения |
| [`patterns/`](./patterns/) | Повторяемые приёмы (API, Prisma, ошибки, клиент) |
| [`decisions/`](./decisions/) | ADR: зафиксированные решения (не нарушать без нового ADR) |
| [`features/`](./features/) | Индекс крупных фич + шаблон карточки фичи |
| [`ui/`](./ui/) | Визуальные и UX-инварианты текущего приложения |

**Точка входа для агента:** корневой [`AGENTS.md`](../AGENTS.md).
