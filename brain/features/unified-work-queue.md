# Единая рабочая очередь

## Цель

Дать сотруднику один экран для проектов, самостоятельных заявок и быстрых расчётов с понятными фильтрами, предпросмотром и безопасными быстрыми действиями.

## Границы первой итерации

- Не объединять таблицы `Project` и `Order`.
- Не менять резерв, выдачу, приёмку и складские транзакции.
- Не импортировать строки из внешнего Excel без предварительного просмотра и подтверждения.
- Не считать `ESTIMATE_ONLY` финансовым фактом.

## Затронуто

- Prisma: `ProjectMode`, nullable `Project.customerId`, `Project.leadCustomerName`, логотип заказчика.
- API: `/api/work-queue`, создание/преобразование расчёта, логотип заказчика.
- UI: `/work`, навигация, админка заказчиков.
- Analytics: отдельный последующий контур сверки внешней таблицы и сайта.

## Инварианты

- Проект со связанными заявками — одна карточка; заявки показываются внутри неё.
- Самостоятельная заявка — отдельная карточка.
- `FULL` проект всегда имеет заказчика.
- Быстрые действия используют разрешённые доменные переходы.
- Архивирование остаётся отдельной операцией.
- Цвет кодирует только состояние: действие Wowstorg, ожидание клиента, склад, риск, завершение.

## Приёмка

- [x] В навигации есть «Рабочая очередь», «Все проекты» и «Все заявки».
- [x] Очередь фильтруется по периоду, типу, фазе, датам и поиску.
- [x] Проекты без дат и расчёты доступны отдельными пресетами.
- [x] Пользователь может сохранять собственные наборы фильтров локально.
- [x] Проекты группируются по месяцу события, записи без дат выделены отдельно.
- [x] Карточка проекта показывает дочерние степперы заявок без дублей.
- [x] Карточки разворачиваются без перехода на детальную страницу.
- [x] В предпросмотре проекта видны смета, задачи, контакты и быстрые действия.
- [x] Статус проекта меняется из очереди; завершение и отмена требуют подтверждения и архивируют проект по существующим доменным правилам.
- [x] «Составить смету» создаёт `ESTIMATE_ONLY`.
- [x] Расчёт можно преобразовать в полноценный проект без потери сметы.
- [x] У заказчика можно загрузить, заменить и удалить логотип.
