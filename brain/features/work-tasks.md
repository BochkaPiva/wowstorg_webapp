# Задачи / рабочая доска

## Цель
Встроить в Wowstorg локальную рабочую доску в стиле YouGile: колонки, задачи, исполнители, дедлайны и подзадачи без внешней синхронизации.

## Границы
- Не интегрируем YouGile API в MVP.
- Не делаем двустороннюю синхронизацию с внешними досками.
- Не заменяем статусы проектов и заявок задачами: задачи дополняют операционную картину.

## Затронуто
- Prisma: `WorkTaskBoard`, `WorkTaskColumn`, `WorkTask`, `WorkTaskChecklistItem`, `WorkTaskPriority`.
- API: `/api/tasks/**`.
- UI: `/tasks`, блок «Мои задачи» на главной.
- Связи: задачи могут ссылаться на `Project`, `Order`, `User`.

## Инварианты
- Задачи доступны только `WOWSTORG` в первом срезе.
- Дедлайны хранятся как date-only через UTC midnight, по текущему паттерну проекта.
- Завершённые задачи скрываются с основной доски и из «Моих задач».
- Архив задач — отдельное мягкое состояние `WorkTask.archivedAt`: архивные задачи не видны на доске, в дашборде, напоминаниях и «Моих задачах», но доступны из скрытого архива доски.
- Нельзя удалить колонку, пока в ней есть задачи.

## Приёмка
- [x] Есть рабочая доска с колонками.
- [x] Можно добавлять, редактировать и удалять пустые колонки.
- [x] Можно добавлять, редактировать, перемещать и удалять задачи.
- [x] У задачи есть исполнитель, дедлайн, приоритет, цвет, проект, заявка.
- [x] У задачи есть подзадачи-чеклист.
- [x] На главной отображаются задачи текущего сотрудника.
- [x] Колонка завершения умеет архивировать выполненные задачи без удаления.
