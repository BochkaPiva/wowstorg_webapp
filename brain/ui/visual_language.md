# Визуальный язык (текущее приложение)

## База

- **Шрифты:** Geist (Sans + Mono) из `next/font`, в `globals.css` для body задан запасной стек **Arial, Helvetica, sans-serif**.
- **Тема:** в комментарии к `globals.css` зафиксирована **светлая тема v1** — не включать тёмную тему без отдельного прохода по контрасту и согласования.

## Цвета и фон

- Фон приложения в **`AppShell`**: мягкий **лавандово-жёлтый градиент** (`#f6f2ff` + radial violet/yellow).
- Акцент навигации: **`violet-700`** для активного пункта, **`violet-50`** hover для неактивного.
- Текст: **`zinc-800`**, вторичный **`zinc-600`**.

## Декоративные элементы

- Классы **`.wow-bg`**, **`.wow-orb`** — анимированные пятна; **`prefers-reduced-motion: reduce`** отключает анимацию — сохранять это при добавлении новых эффектов.

## Формы и плотность

- Навигационные ссылки: **`rounded-lg px-3 py-2 text-sm`**.
- Новые блоки стараться визуально **согласовывать скругление и отступы** с соседними карточками в том же экране (не вводить новый радиус без причины).
